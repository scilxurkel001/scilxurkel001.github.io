逐火十三英桀档案

「救世」凯文

我是凯文，持有「救世」之铭的战士。除此之外，再无其它了。

简介： 往日的传说无需赘述，他是最接近「逐火之蛾」宏愿的人，被所有人承认的「英雄」。
世人坚信，他终将带领人类战胜崩坏。




「■ ■」爱莉希雅

“如你所见，与那个凯文齐名的第二领袖，竟是一位如花朵般娇羞的少女——也就是我啦。”

简介： 凡事都听凭心意而为，自由自在，与「第二位」的身份格格不入的少女。英桀们的另一位领袖。
无论是引起她的注意，亦或是令她失去兴趣，都值得为之如临如履。




「戒律」 阿波尼亚

“如同预言中那样，人们所畏惧的事物，被我一一带来，一一实现。这无法托辞于命运的安排，我知我罪，我罪常在我前。”

简介：
不可知不可觉的丝线，将诸人的命运紧紧缠缚。
它引导着生命，在啼哭中降生；它轻挽着灵魂，于沉寂中归去。
终局已定，一如阿波尼亚目中所见；囚牢已成，唯有阿波尼亚身陷其中。




「黄金」伊甸

“我的时代已经一曲终了，愿你们的未来能够长颂不休。”

简介：
直至那场大火将万物付之一炬前，始终作为耀眼的「巨星」为世界所知的女子。
在这日渐沦亡的末世，她仍未停止放歌，以金色的声调，使人们忆起旧时代的烟尘。




「螺旋」 维尔薇

“没错，就是我。螺旋工坊最年轻的主人，对凯文武装的发明者、黄昏纪元的魔术王、俯瞰群星之人，以及——热爱制造「天才小垃圾」 的维尔薇！”

简介：
维尔薇热衷于用生活表演她最挚爱的「魔术」——齿轮转动，拉开舞台的幕布；蒸汽喧闹，献上不绝的欢呼。
身为英桀们的工程顾问，她造就着「常识」与「反常识」的螺旋，收获的赞誉与质疑也同发色一般分明对立。




「鏖灭」千劫

"敌人，杀便好。拦路者，杀便好。存在的隐患，杀便好。所有的异物，杀便好……！"

简介：
以漆黑的面具遮蔽真容，持有「非天」之异名的男子，亦是秘密行动部队「毒蛹」的一员。
因其难以相处的乖戾性情与桀骜不驯的姿态，仅位列第六。




「天慧」苏

“我只是一介旁观者，前方的路将往何去，你自有答案。”

简介：
以医师的身份入世，以仁心为兆始开悟。
洞观三千世界，这位始终闭目思量的智者，终将成为一瞬天慧的权化。




「刹那」樱

“对待敌人无需怜悯，那是对自己的残忍。但……你必须要认清，谁才是真正的敌人。”

简介：
故国崩陷，万里独行，一场不为人知的交易，使她成为了逐火的飞蛾。
身为秘密行动部队「毒蛹」的一员，她的剑生于阴影，自始自终指向最为特殊的敌人。




「旭光」科斯魔

“我无法给予你指引，除非你想要的是来自失败者的教训。「旭光」之途是从无数条末路中走出的小径……就只是这样而已。”

简介：
光阴在科斯魔的身后留下三道影子——孩童、少年与英雄。
身为孩童的时光，早已远去。而化身英雄的梦想，却又已然无法实现。
漫漫长夜，名为「科斯魔」的少年独行于此，如若一缕旭光，静候黎明。




「无限」梅比乌斯

"进化的过程并不美丽。正相反，它非常丑陋，丑陋至极。但只有经历过这个丑陋的过程，我们才能真正理解……生命的美丽"

简介：
正如那对深不见底的蛇瞳，「女孩」的真正面目罕有人知。
那双眼睛见证了崩坏一次又一次降临。在进化的路途上，为穷尽世间真理，她从不介意任何代价。




「繁星」 格蕾修

“和我一起画吧，这样，它们也是你的朋友了。你喜欢什么？海底的帆船？还是云朵里的螃蟹？”

简介：
格蕾修不爱言语，所思所想皆在画笔之下；格蕾修不闻世事，心之所向尽在色彩之中。
一幅画卷，织成了格蕾修与她的世界。往日的色彩，于绘梦之中铺展，直至银海的尽头。




「浮生」华

“我还是我，在成为未来的我之前，还有一条距离未知的路要走。命运只有一种，谁也无法绕开。”

简介：
沉默少言的年轻战士。彼时的少女青涩、迷茫，尚未预见自己明日多舛的命途。
未来永劫，她如一羽雏鸟，失去了来时的道路，亦不知去路的方向。




「空梦」帕朵菲莉丝

“是呀，大家一般都不会用全名称呼我。有人叫我菲莉丝，也有人叫我帕朵。怎么都可以啦，你觉得顺口就好。毕竟咱跟劫哥蛇姐不一样，是个很好说话的人呀！”

简介：
人生不过一场随机的游戏——一场博弈，一次掷骰，令她脱胎换骨，踏上了逐火的旅路。
身为十三英桀末席，其地位与实力同其余十二人隔有天壤之别（自称）。
